# Website Pariwisata Yogyakarta

Website pariwisata Yogyakarta merupakan sistem informasi pariwisata yang ada di kota Yogyakarta. Website ini dikembangkan dengan PHP native.

## Features

- Login
- Register
- Logout
- Dasboard Admin
- CRUD data di halaman admin

## System Requirements

- PHP => 7.4.15
- MySQL => 5.7.33
- XAMPP
- Visual studio code
